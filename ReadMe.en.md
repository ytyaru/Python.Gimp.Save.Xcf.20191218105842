# Python.Gimp.Save.Xcf

　Save xcf file with GIMP Python-fu.

# Usage

```sh
git clone https://github.com/ytyaru/Python.Gimp.Save.Xcf.20191218105842
cd Python.Gimp.Save.Xcf.20191218105842/src
./run.sh
```
